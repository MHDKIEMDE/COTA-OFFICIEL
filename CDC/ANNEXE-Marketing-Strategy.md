## 📢 CANAUX D'ACQUISITION

### Phase 1 - Acquisition gratuite (Mois 1-4)

#### 1. Bouche-à-oreille (40% des users)

**Programme parrainage** :
- 1 filleul = 3 jours premium
- 5 filleuls = 7 jours
- 10 filleuls = 30 jours
- 20 filleuls = Premium à vie 🏆

**Cashback parrainage** : 10% sur achats des filleuls (Phase 2)

#### 2. Groupes WhatsApp / Facebook foot (25%)

**Stratégie** :
- Identifier 50+ groupes WhatsApp/FB d'amateurs foot
- Partager pronostics gratuits 3-4 fois/semaine
- Toujours avec analyse + lien app
- **Pas de spam** : valeur d'abord, vente ensuite

**Format** :
```
🏆 Mon prono du jour : PSG vs OM

✅ Forme PSG : 5V/5
✅ H2H favorable
✅ Joueurs clés présents
⚠️ Match à risque (rivalité)

Ma prédiction : PSG gagne (cote ~1.45)

Plus de pronos sur FootApp 👇
[lien]
```

#### 3. SEO / Site web (15%)

Voir `MODULE-Site-Web-SEO.md`. Cibles :
- 1000+ visiteurs SEO/mois en 6 mois
- Articles longue traîne (FAQ, guides)

#### 4. Telegram channels (10%)

**Stratégie** :
- Créer un channel Telegram FootApp
- Publier 2-3 pronos gratuits/jour
- Lien vers app pour pronos premium
- 1 000+ abonnés à 6 mois

#### 5. YouTube Shorts / TikTok (10%)

**Format** :
- Vidéos 30s : "Mon prono du jour avec analyse"
- Stats animées
- CTA : "App dans la bio"

**Fréquence** : 1-2 vidéos/jour pendant 3 mois → 5k abonnés/mois.

### Phase 2 - Acquisition payante (Mois 5-10)

#### 1. Facebook / Instagram Ads (40% budget)

**Budget** : 200-500€/mois

**Cibles** :
- Hommes 22-40
- Pays : Sénégal, CI, Burkina, Mali
- Centres d'intérêt : foot, paris, statistiques
- Comportements : utilisateurs Mobile Money

**Formats** :
- Vidéos 15s avec hook fort
- Carrousel (3 slides : problème, solution, CTA)
- Stories : avant/après ("Ces gens ont gagné avec FootApp")

**A/B test** :
- 5 hooks différents
- Optimiser 2 fois/semaine
- Cibler CPI < 0.30€

#### 2. Google Ads (20% budget)

**Budget** : 100-200€/mois

**Mots-clés** :
- "application pronostic foot"
- "pronostic foot afrique"
- "betwinner pronostic"
- "1xbet astuces"

**Type** :
- Search ads (intention forte)
- Universal App Campaigns (auto)

#### 3. Influenceurs micro/macro (30% budget)

**Budget** : 200-400€/mois

**Stratégie** :
- 5 micro-influenceurs (5-50k followers) à 50-100€/post
- 1 macro-influenceur (>100k) à 300-500€/post

**Cibles influenceurs** :
- Comédiens TikTok foot
- Analystes foot YouTube
- Streamers FIFA Twitch
- Tipsters Telegram (avec audience)

**Format** :
- Avis honnête (pas pub déguisée)
- Code promo unique (tracking conversions)
- Stories Instagram en plus du post principal

#### 4. Partenariats clubs amateurs (10% budget)

**Budget** : 100€/mois

**Concept** : sponsoriser des équipes de quartier
- 50€ pour un maillot (logo FootApp)
- Annonce dans le club (parrainage joueurs)
- Engagement très fort dans la communauté

## 🎯 CAC / LTV

### Calcul CAC (Cost Acquisition Customer)

```
CAC = Total dépenses marketing / Nombre nouveaux clients payants

Exemple Phase 2 :
Budget : 680€/mois
Conversions : 100 abonnements
CAC = 6.80€
```

### Calcul LTV (Lifetime Value)

```
LTV = Revenu moyen par user × Durée moyenne abonnement

Exemple :
- Mensuel : 8000 FCFA × 3 mois moyen = 24 000 FCFA (~36€)
- Trimestriel : 20 000 FCFA × 1.5 cycle = 30 000 FCFA (~45€)
- VIP : 15 000 FCFA × 6 mois = 90 000 FCFA (~135€)

LTV moyen pondéré : ~50€
```

### Ratio LTV / CAC

```
LTV / CAC = 50 / 6.80 = 7.35

Cible saine : > 3
Excellent : > 5
FootApp : 7+ ✅
```

## 💡 RAPPELS CLÉS

1. **La data avant le marketing** : si l'algo n'atteint pas 55%, le marketing ne sauvera pas
2. **Les premiers users sont précieux** : leur feedback >>> nouveaux users
3. **Acquisition gratuite > payante** au début
4. **Mesurer tout** : sans data, pas d'optimisation
5. **Patience** : 6-12 mois pour un produit qui scale
